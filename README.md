# Advanced Portfolio

#### How to launch:

1. ```git clone```
2. ```yarn``` / ```npm i```
3. ```yarn global add gulp-cli``` / ```npm i gulp-cli -g```
4. ```gulp```
5. Navigate to your localhost address
(default is http://localhost:3000/)

An advanced version of the portfolio, designed by © Loftschool.
